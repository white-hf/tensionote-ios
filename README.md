# Tensionote iOS

iOS native app for Tensionote.

## Tech Stack

- Swift
- SwiftUI
- Local storage
- Local reminders
- PDF export
- Email/share flow

## Notes

- Supports Simplified Chinese, English, and Hindi
- V1 is local-only storage
- Home screen is focused on quick entry and last 2 weeks trend
- This repository is prepared from Windows; final Xcode build validation should be done on macOS
